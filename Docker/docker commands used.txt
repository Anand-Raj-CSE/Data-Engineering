1. To build images 
    docker build -t welcome .   ---> Here . signifies all the files to be read , welcome is image name.
2. See all docker images
    docker images
3. Pull any docker image from docker hub
    docker pull docker/getting-started
4.  run any container
    docker run -d -p 80:80 docker/getting-started
5. check all the runnig containers
    docker ps
6. clear screen ---> cls
7. Stop container 
    docker stop container id
